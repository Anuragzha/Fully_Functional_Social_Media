# Fully_Functional_Social_Media
AnuLink is a futuristic social media platform designed to connect people in a more engaging and dynamic way. Built using Django, it offers core social networking features like user authentication, post creation, following system, and personalized feed suggestions.

Key Features:
✅ User Authentication – Secure sign-up and login system.
✅ Post Creation & Media Upload – Share text, images, and videos.
✅ Follow System – Users can follow/unfollow others.
✅ Engaging Feed – Personalized posts from followed users.
✅ Like & Comment System – Interactive engagement features.
✅ Follow Suggestions – Smart recommendations for connections.
✅ Admin Panel – Manage users and content efficiently.

Technology Stack:

    Backend: Django (Python)

    Frontend: HTML, CSS, Bootstrap, SCSS, JavaScript

    Database: SQLite (also can be scaled to PostgreSQL)

    Authentication: Django Auth System

    Media Handling: Django ImageField & Pillow

AnuLink aims to redefine online interaction by fostering meaningful digital connections while maintaining a smooth and modern user experience.
